# ⛅ Weather Forecast Web App

A modern and lightweight weather forecasting application that allows users to view real-time weather conditions for any city worldwide. Built using **HTML5, CSS3, and JavaScript**, it utilizes the **OpenWeatherMap API** to fetch and display accurate weather information.

## 🚀 Features

* 🌍 **City-Based Weather Search**
  Enter any city name to get the current weather conditions.

* 🌡️ **Live Weather Details**
  Displays real-time temperature, humidity, wind speed, and dynamic weather icon.

* 🔁 **Dynamic Icon Updates**
  Automatically switches weather icons based on conditions like Cloudy, Clear, Rainy, etc.

* 📱 **Responsive UI**
  Mobile-friendly and visually appealing design with CSS3 gradients and hover effects.

## ⚙️ How It Works

1. The user inputs a **city name** into the search field.
2. On clicking the **search button**, an API request is sent to OpenWeatherMap.
3. The app receives JSON weather data including:

   * Temperature
   * Humidity
   * Wind speed
   * Weather condition
4. UI updates dynamically:

   * Displays numerical data
   * Updates weather icon based on condition (e.g., Cloudy, Rainy)
